Kayak VR Experience

Kayak VR Simulator (Unity | Google CardBoard | C#)
Game Developer | Utopic Studio Spa
  • Game development of a Kayak VR Experience, including repository setup and build deliverables.
  • The team was remote, so work was coordinated using a dedicated Slack workspace.
  • Built an infinitely scrolling map system using pre-built scenery pieces that are dynamically spawned based on the player's travelled distance.
    • Map segments to be spawned have a weight to control how often we can spawn the same segment.
    • Each map segment has a spline path, which is used to move the kayak across the map terrain.
    • Obstacles are randomly placed over each segment using grid positioning.
    • A custom tool was also built to control this system.
  • The project uses Steam Audio for binaural audio spatialisation, but additional logic was added to control the audio in-game.
    • Each obstacle has a custom component that updates the AudioSource's priority based on its distance from the listener.
    • For the waterfalls, the AudioSource was set to 2D, but with dynamic volume control based on how far they were from the listener. This was done because by default 3D audio comes from a single point, but a more global sound was needed for the waterfalls.
    • Custom inspectors and scene_gui tools were added to better debug the dynamic behaviour of audio sources in the game.
Addittionally, I also added the sound effects for this game.
