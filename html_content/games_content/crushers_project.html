Crushers!

Casual match-3 multiplayer game with RPG elements (Unity | C# | Photon Quantum)
Senior Game Developer | Contractor for Redemption Games Inc.
  • Prototyped key gameplay features, including the Home panel tabs, Post Level Win/Lose sequence, and Trophy Road, helping shape core game mechanics.
  • Developed shared libraries for core features such as UI management, assets retrieval, and object pooling, enabling code reuse across multiple projects and improving development efficiency.
  • Implemented game configuration systems for features like Trophy Road and the main Core-Loop, ensuring seamless synchronization between backend and client. Used the same approach to define in-game items and store products, ensuring data integrity across all features.
  • Designed and implemented a flexible, data-driven, and event-driven Tutorial System with live data updates for rapid debugging and iteration. The system, JSON-based for remote updates, supported custom actions such as navigating tabs, displaying custom assets, and triggering dialogues. It featured checkpoint and persistence mechanisms, allowing players to resume tutorials from saved progress and ensuring continuity even if they leave mid-tutorial.
  • Developed a robust audio system with SFX variation, pitch/volume randomization, audio ducking, and dedicated buses for music and SFX. The system dynamically adapted to game and pause states using different AudioMixer presets.

Additionally, prototyped initial versions of key gameplay features, including GoldRush, Season Pass, and Weekly Events, contributing to player engagement and monetization strategies.
