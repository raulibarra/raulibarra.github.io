StarWars Galactic Defense

Tower Defense Game developed (Unity | C#)
Senior Game Developer | DeNa Santiago
  • Developed and expanded a data-driven abilities system for towers and characters, integrating instant stat changes (e.g., healing), status effects (e.g., damage over time), and area-of-effect mechanics (e.g., healing fountains, lava zones), enriching gameplay and player interaction.
  • Optimized Android game size by 35% using ETC compression for transparent textures. Developed a pipeline to split textures into RGB and Alpha channels and created shader variants to recombine them at runtime, reducing memory usage and improving performance on OpenGL 2.0 devices.
  • Collaborated with artists to integrate VFX and animations, ensuring seamless synchronization between visual elements and game logic. Contributed to social features, including Facebook friend requests and energy-sharing.
  • Developed editor tools to streamline workflows, including a map editor, UI skinning system, and HotReload with real-time stats debugging for designers.
  • Programmed an audio system with separate controls for music and SFX, prioritizing key audio events and implementing audio ducking, pre-dating Unity's audio mixer.
