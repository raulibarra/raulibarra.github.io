Asteroids New Hope

2D Asteroids game (Unity | C# | D.O.T.S.)
  • Created mechanics for aiming, shooting, firing projectiles, destroying asteroids and spawning impact VFXs.
  • Created logic to randomly spawn weapon upgrades when destroying UFOs, such as increase fire rate or fire more bullets at once.
  • Added basic player stats system to keep track of lives left, enemies destroyed and upgrades equipped.
  • Core audio manager to play music and trigger sfxs using Unity's built-in system (it uses a pool of audio sources for better performance).
  • Simple but effective bullet pooling system based on Unity's pooling classes.
  • Created logic to move asteroids and bullets using D.O.T.S. system.
  • Created a basic HUD system to display your score, remaining lives and stats when you lose (i.e. asteroids/enemies destroyed).

Additionally, I composed the music and designed sound effects for the game.
