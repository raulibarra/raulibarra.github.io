Asteroids New Hope

2D Asteroids game (Unity | C# | D.O.T.S.)
  • Created mechanics for aiming, shooting, firing projectiles, destroying asteroids and spawning impact VFXs.
  • Created logic to randomly spawn "upgrades" when destroying UFOs such as "increasing fire ratio" or "firing more bullets" at once.
  • Added basic player stats system to keep track of lives left, enemies destroyed and equipped upgrades.
  • Core audio manager to play music and trigger sfxs using Unity's builtin system (It uses a pool of AudioSources for better performance).
  • Basic but effective pooling system for bullets based on Unity's Pooling classes.
  • Created logic to move asteroids and bullets using D.O.T.S. system.
  • Created a basic HUD system to display your score, remaining lives and stats when you lose (ie: asteroids/enemies destroyed).

Additionally, I composed the music and designed sound effects for the game.
