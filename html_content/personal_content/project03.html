2D Funky Platform Demo

2D Platform game (Unity | C#)
  • Core gameplay mechanics (jumping, swimming, falling rocks, coins collection, health tracking).
  • Added camera movement (2D confinement, smooth tracking and position based camera shakes using Cinemachine.
  • Added in-game sequences using Unity's timeline.
  • Added basic player stats system to keep track of your remaining lives and score.
  • Added simple system (using Unity Physics) to simulate the player being dragged by river's movement.
  • Created a simple dialogue system that teaches the player how to play and advance.
  • Core audio manager to play music and trigger SFXs using Unity's builtin system (It uses a pool of AudioSources for better performance).
  • Simple intro-loop music system for the in-game music.

Additionally, I composed the music and designed sound effects for the game.
