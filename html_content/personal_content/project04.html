Slippery Logs (Unity | C# | PlayFab | NodeJS)

Infinite Jumper with remote settings, leaderboard and backend persistence
  • Core gameplay logic (jumping over logs, character energy, collectables and enemies).
  • Added basic player stats system to keep track of your remaining energy and score.
  • Integrated PlayFab to store player data and retrieve leaderboards.
    • Uses custom functions to increase player score based on remote parameters.
    • While scores are simulated locally during jumps, they are calculated and retrieved from the backend after X number of jumps to prevent cheating.
    • All parameters are stored and retrieved from PlayFab
    • Localization data is also retrieved from PlayFab.
  • Created a simple but effective pooling system for logs and collectables.
  • Created a system that spawns log sections infinitely as you progress.
  • Core audio manager for playing music and triggering SFXs using Unity's built-in system (it uses a pool of audio sources for better performance).
  • Created and programmed all the game's screens.

I also composed the music and designed the sound effects for the game.
