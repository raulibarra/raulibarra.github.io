Simple FPS & MetaSounds Demo

3D FPS Game (Unreal | C++ | Blueprints | MetaSounds)
  • Created mechanics for aiming, shooting, firing projectiles and spawning impact VFXs.
  • Added basic character stats system to keep track of damage and being able to die or kill enemies.
  • Integrated animations with event notifiers to fire footstep SFXs.
  • Designed enemies AI using Unreal's Behaviour trees to make them chase/fire the player.
  • Created a basic HUD system to display your health and opponents remaining count.
  • Designed a simple but effective interactive music system using MetaSounds, that switches to a more intense music when fighting enemies.
  • Most of the code is written in C++

Additionally, I composed the music and designed sound effects for the game.
